# About VTS

## Overview

This guide will help you deploy, test, and use the Argus Video Training System (VTS). Originally developed for training underground heavy vehicle operators remotely, Argus works equally well for any situation where you need live video feeds deployed quickly. Everything you need comes in durable hard cases designed for fast setup anywhere.

## System Key Components

The Argus Video Training System comprises of four main components:

| Component | Description |
|-----------|-------------|
| **The Argus Software** | Windows-based software platform that gives remote trainers access to multiple video feeds from remote cameras |
| **The Argus Access Points (AP)** | Wireless networking nodes that work together to create a mesh network |
| **Remote Camera Systems (RCS)** | Portable video streaming cameras that quickly attach to training vehicles |
| **Trainers Tablet PC** | Connects to the access points to view camera feeds and save recordings |

## Video Training System

The Video Training System kit comprises of:

The kit includes:

- Multiple wireless access point units for mesh network creation
- Remote camera systems (RCS) for video capture
- Trainer tablet PC for viewing and controlling video feeds  
- Durable hard cases for transport and protection
- Milwaukee battery packs (18V/6Ah for APs, 12V/3Ah for RCS)
- Charging equipment
- Mounting accessories (magnetic mounts, suction cups)

Each component is designed for reliable operation in challenging environments including underground locations where standard Wi-Fi isn't available.

## Key Features

### Wireless Mesh Network
- Automatic connection and configuration
- Multiple access points work together seamlessly
- Extends coverage over large areas
- No cables or fixed infrastructure required

### Multiple Video Feeds
- View up to 4 camera feeds simultaneously
- Toggle between feeds in real-time
- Record all streams simultaneously for later review

### Battery Powered
- Standard 18V Milwaukee batteries for access points (minimum 10 hours operation)
- Standard 12V Milwaukee batteries for cameras (12 hours operation)
- Swap batteries between training sessions for continuous operation

### Rapid Deployment
- Minimal setup time
- Intuitive configuration
- Fast network mesh establishment
