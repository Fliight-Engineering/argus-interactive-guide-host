# Trainers Tablet

The trainers tablet connects to the access points to view camera feeds and save recordings.

<div class="image-wrapper-70">
![Trainers Tablet](/img/vts-guide/trainer-tablet.png)
*Trainer tablet device showing the Windows-based interface*
</div>

<div class="highlights-grid-single">

<div class="highlight-card">
<div class="card-content">

### Capabilities

- **Live Video Viewing** - View camera feeds from all connected cameras
- **Multi-Stream Recording** - Record all video streams simultaneously
- **Network Monitoring** - Monitor mesh network health and connectivity status
- **System Control** - Manage system settings and configuration
- **Recording Management** - Access and playback saved recordings

</div>
</div>

<div class="highlight-card">
<div class="card-content">

### Specifications

- **Operating System**: Windows-based tablet PC
- **Display Orientation**: Landscape mode (cameras at top edge when horizontal)
- **WiFi Connection**: Connects to ARGUS-VTS-XXX network
- **Power**: Built-in battery with external charging
- **Interface**: Touchscreen with Windows interface

</div>
</div>

<div class="highlight-card">
<div class="card-content">

### Connection

The tablet connects wirelessly to the access point mesh network via WiFi:

- **Network Name**: ARGUS-VTS-XXX (where XXX is the kit number)
- **Password**: BarmincoVTS
- **Range**: Up to 50 meters with line of sight
- **Connection**: Automatic once WiFi credentials are entered

</div>
</div>

<div class="highlight-card">
<div class="card-content">

### Operation

The tablet runs the Argus software which provides:

- Real-time video feeds from all connected cameras
- Ability to toggle between camera views
- Simultaneous recording of all camera streams
- Network health monitoring indicators
- Playback of recorded training sessions

</div>
</div>

</div>

For detailed setup instructions, see [Tablet Setup](/docs/setup-operation/tablet-setup).
