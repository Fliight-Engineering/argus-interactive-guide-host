# Argus VTS Guide
# User Guide

Welcome to the Argus Video Training System user guide. This guide will help you deploy, test, and use the Argus VTS.

Originally developed for training underground heavy vehicle operators remotely, Argus works equally well for any situation where you need live video feeds deployed quickly. Everything you need comes in durable hard cases designed for fast setup anywhere.
