# Argus Video Training System (VTS) - User Guide

Welcome to the Argus Video Training System user guide. This guide will help you deploy, test, and use the Argus VTS.

Originally developed for training underground heavy vehicle operators remotely, Argus works equally well for any situation where you need live video feeds deployed quickly. Everything you need comes in durable hard cases designed for fast setup anywhere.

## System Highlights

✅ **Rapid Deployment** - Setup in minutes, not hours  
✅ **Wireless Mesh Network** - Automatic connection, no cables needed  
✅ **Multiple Video Feeds** - View and record from up to 4 cameras simultaneously  
✅ **Reliable Performance** - Proven underground and remote location operation  
✅ **Battery Powered** - Everything runs on standard batteries for maximum portability  

## Getting Started

Start by reviewing the [About VTS](/docs/about-vts) section to understand the system components, then follow the [Setup & Operation](/docs/setup-operation) guide for your first deployment.
