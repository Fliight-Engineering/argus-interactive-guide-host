## Power up tablet

On the top edge of the PC, on the left-hand side, there are three silicon buttons. The **leftmost is the power button** for turning the trainers tablet PC on and off.

<div class="tablet-image-container">

![Tablet Power Button Location](/img/vts-guide/tablet-buttons.png)

</div>

- **Leftmost button** = Power button (Power on/off)
- **Middle button** = Volume Up
- **Rightmost button** = Volume Down

## Logging In

### PIN Login

To log in to the tablet:

1. **Swipe up** from the lock screen to show the Argus VTS user login page
2. **Tap the PIN input field**
3. **Enter the Windows Tablet PIN**: `2580`
4. **Press Enter** or tap OK

The desktop will then be ready for use.

:::info Windows Tablet PIN
The default PIN for the Argus VTS tablet is: **2580**
:::

### After Login

Once logged in:

1. Desktop should be visible
2. Argus software should be available (check taskbar)
3. WiFi connection options will be available
4. Ready to connect to AP-1 WiFi

- **WiFi Name**: `ARGUS-VTS-XXX*` *(XXX being the kit number)*
- **Password**: `BarmincoVTS`
