# Tablet Setup

## Tablet PC Initialization

When in landscape mode, the trainer tablet PC's cameras will be located at the top of the screen.

![Tablet Camera Location](/img/vts-guide/image7.png)

*Tablet PC showing camera location at top edge when in landscape orientation*

### Initial Setup

The tablet comes pre-configured, but if you need to reinitialize:

1. Power off completely if needed
2. Power on using the power button (top edge, left-hand side)
3. Wait for Windows to load (approximately 30 seconds)
4. Unlock the screen if needed

## Power Up Tablet

On the top edge of the PC, on the left-hand side, there are three silicon buttons. The **leftmost is the power button** for turning the trainers tablet PC on and off.

![Tablet Power Button Location](/img/vts-guide/image13.png)

*Top edge of tablet showing power button location (left-hand side)*

- **Leftmost button** = Power button (Power on/off)
- **Middle button** = Volume Up
- **Rightmost button** = Volume Down

### Power On Steps

1. Locate power button on top left edge
2. Press firmly for 1-2 seconds
3. Release and wait for boot sequence
4. Wait for desktop to appear
5. Proceed to Logging In

## Logging In

### PIN Login

To log in to the tablet:

1. **Swipe up** from the lock screen to show the Argus VTS user login page
2. **Tap the PIN input field**
3. **Enter the Windows Tablet PIN**: `2580`
4. **Press Enter** or tap OK

The desktop will then be ready for use.

:::info Windows Tablet PIN
The default PIN for the Argus VTS tablet is: **2580**
:::

### After Login

Once logged in:

1. Desktop should be visible
2. Argus software should be available (check taskbar)
3. WiFi connection options will be available
4. Ready to connect to AP-1 WiFi

## WiFi Connection

### Connect to AP-1

The trainer access point (AP-1) will allow the trainer tablet to auto-connect to its WiFi hotspot for access. It should be the closest access point to the trainer's safe location.

:::note WiFi Performance
WiFi performance is excellent with a range of up to **50 meters** with line of sight.
:::

**AP-1 WiFi Details:**
- **WiFi Name**: `ARGUS-VTS-XXX*` *(XXX is the kit number)*
- **Password**: `BarmincoVTS`

### Manual Connection (if needed)

1. Open WiFi settings
2. Look for network: **ARGUS-VTS-XXX**
3. Select the network
4. Enter password: **BarmincoVTS**
5. Wait for connection (should happen automatically)

### Verify Connection

- WiFi icon shows connected status
- Network name appears in connection list
- Argus software shows network connectivity

:::note
**Position**: The trainer tablet should be positioned as close to the AP-1 access point as practical (within 50 meters) for best WiFi performance.
:::

## Argus Software Launch

Once connected:

1. Launch Argus software (usually pinned to taskbar or start menu)
2. Wait for software to initialize (10-15 seconds)
3. Software should automatically detect connected cameras
4. All 4 camera feeds should start streaming automatically
5. Check Network Health Monitoring Panel (lower right)

## Key Touchscreen Tips

- **Landscape Orientation**: Use tablet in landscape for optimal viewing
- **Double-tap Video**: Double-tap any sidebar video to view full-screen
- **Settings Button**: Access settings for configuration and playback
- **Record Button**: Start/stop recording all connected camera feeds
